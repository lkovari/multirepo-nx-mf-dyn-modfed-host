About This Page

This is an independent micro-frontend host application. It can be developed, tested, and deployed separately from the remotes. The remotes (Remote A, B, C) are integrated seamlessly using Dynamic Module Federation at runtime.

Features

  • Standalone Angular components
  • OnPush change detection
  • Modern Angular v21 best practices
  • Dynamic module federation
  • Independent deployment (multi-repo)
